# 🚍 COUNTBUS – Sistema Inteligente de Contagem de Passageiros

Uma solução moderna para monitoramento e análise do fluxo de passageiros em transportes públicos e privados.

---

## 📌 Sobre o Projeto

O COUNTBUS é uma plataforma que integra hardware e software para realizar a contagem automática de passageiros em tempo real. Sensores instalados no veículo capturam os dados e enviam para um servidor, onde podem ser visualizados através de uma interface web. 
<br/> Para mais informações, <a href="https://countbus.github.io"> acesse nossa página </a>.

---

## 🎯 Objetivo

- Melhorar a gestão de transporte  
- Otimizar rotas e horários  
- Reduzir custos operacionais  
- Apoiar decisões com base em dados reais  

---

## ⚙️ Como Funciona

1. Sensores detectam entrada e saída de passageiros  
2. Um microcontrolador processa os dados  
3. As informações são enviadas para o backend via API  
4. Os dados são armazenados e exibidos em uma dashboard  

---

## 🧩 Tecnologias Utilizadas

### 💻 Backend
